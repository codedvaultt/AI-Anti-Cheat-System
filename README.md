# 🛡️ ProctorGuard – AI Anti-Cheating System

## 📌 Overview

ProctorGuard is a web-based AI anti-cheating system designed to monitor users during online examinations. It uses browser-based monitoring and simulated computer vision techniques to detect suspicious activities in real time and ensure exam integrity.

---

## 🚀 Features

* 🎥 Real-time webcam monitoring
* 👤 Face detection simulation (absence / multiple faces)
* 👁️ Gaze tracking simulation (looking away detection)
* 🗂️ Tab switching detection
* 📋 Copy–paste activity tracking
* 🖱️ Right-click restriction
* 🔧 Developer tools detection (F12, inspect)
* 📸 Snapshot capture during violations
* 🚨 Real-time alerts and notifications
* 📊 Risk score calculation
* 📝 Manual & automatic reporting system

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Concepts Used:**

  * Browser event handling
  * DOM manipulation
  * Real-time monitoring logic
  * UI/UX design for proctoring systems

---

## ⚙️ How It Works

1. User starts monitoring session
2. Camera access is requested
3. System tracks user behavior through:

   * Webcam feed (simulated detection)
   * Browser events (tab switch, copy, etc.)
4. Suspicious actions trigger:

   * Alerts
   * Risk score increase
   * Logs & reports
5. Reports can be sent automatically or manually
